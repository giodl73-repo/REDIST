## What is bisect?

**bisect** is a high-performance congressional redistricting platform implementing 20+ algorithms across three algorithmic layers:

- **Structure**: METIS bisection, ApportionRegions, GeoSection, AreaSection, CVD, BFS Growth, Moving-Knife, ILP
- **Search**: ConvergenceSweep, Short-Burst, Forest ReCom, Merge-Split, SMC, Parallel Tempering, VRA-Aware, and more
- **Weights**: geographic, county-weighted, VRA-aligned

Built in Rust. ~2,500× faster than Python GerryChain. 1,500+ tests.

> **Note**: This repository was renamed from `redist` to `bisect` in May 2026 to avoid
> confusion with the R [`redist` package](https://alarm-redist.org/redist/) by McCartan,
> Imai, Kenny & Fifield (winner of the 2022 Statistical Software Award).
